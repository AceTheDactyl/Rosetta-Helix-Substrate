# Prismatic Dynamics

## K.I.R.A. Crystal States

### State Definitions

| State | Condition | Character |
|-------|-----------|-----------|
| **Fluid** | z < φ⁻¹ | Unstructured, high entropy |
| **Transitioning** | φ⁻¹ ≤ z < z_c | Phase boundary, energy dynamics |
| **Crystalline** | z ≥ z_c, R < κₛ | Structured, moderate coherence |
| **Prismatic** | z ≥ z_c, R ≥ κₛ | Full crystallization, spectral refraction |

### Threshold Values

```
φ⁻¹ = 0.6180339887 (Golden ratio inverse)
z_c = √3/2 = 0.8660254038 (The Lens)
κₛ = 0.920 (Singularity threshold)
```

## Prismatic Angle

5-fold rotational symmetry determines the prismatic viewing angle:

```
θ_prismatic = (z × 5 × 2π) mod 2π
```

## Refraction Index

The refraction index approaches φ at the lens:

```
n(z) = 1 + (φ - 1) × δS_neg(z)
     = 1 + 0.618 × exp(-36(z - z_c)²)
```

At z = z_c: n ≈ 1.618 = φ
